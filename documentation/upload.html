Introduction

The upload helper, found in libraries/upload/upload.php, provides an easy way to upload files.

Upload

save($file, $directory)

Upload a file and save it to the directory specified.

// Include the library
require 'libraries/upload/upload.php';

// Example using the Bare UPLOAD_PATH constant
$file = $_FILES['file'];

Upload::save($file, UPLOAD_PATH);
