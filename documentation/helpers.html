Introduction

Several useful helper functions are provided in libraries/helpers.php. This file is loaded automatically by Bare.

Helpers

e($string, [$return = false])

Helper to convert special characters to HTML entities and echo (or return) the result.

// echo
$name = $_GET['name'];
e('Hello, ' . $name);

// return
$name = e($_GET['name'], true);

array_get($key, $array, [$default = null])

Get an item from an array. If the item doesn't exist return a default value.

$array = (
  'name' => 'Chris Hayes',
  'position' => 'Web Developer'
);

$name = array_get('name', $array, 'Sorry, no name.'); // Chris Hayes

load_view($name)

Load a view from the view directory.

load_library($name)

Load a library from the libraries directory.

load_class($name)

Load a class from the classes directory.

redirect($url, [$responseCode = null])

Redirect to a URL.

request_path($path)

Get the requested path relative to the BASE_PATH.

asset($path, [$return = false])

Get a path to an asset relative to the root.

// Echo a path to an asset
<link rel="stylesheet" type="text/css" href="<?php asset('css/style.css') ?>">

// Get a path to an asset
$path = asset('js/app.js', true);

url($path, [$return = false])

Get a path to a url relative to the root.

make_hash($value, [$salt = null])

Create a SHA-256 hash of a value with an added salt.

dd($value)

Dump the given value and kill the script.

str_starts_with($haystack, $needle)

Determine if a given string begins with a given value.

str_ends_with($haystack, $needle)

Determine if a given string ends with a given value.

str_contains($haystack, $needle)

Determine if a given string contains a given sub-string.

str_append($value, $cap)

value a string with a single instance of the given string.

str_prepend($value, $cap)

Cap a string with a single instance of the given string.
